# Dots

This is my configuration, or dotfiles, used to configure UNIX systems in an instant to work the way I like them.
Included are zsh, vim and git configurations with useful aliases, functions and keymaps to improve the way I work.
The setup script automates the configuration process and installs a few tools I enjoy using.

## Usage

Run the following in your terminal emulator:

```sh
source <(curl https://raw.githubusercontent.com/maclong9/dots/refs/heads/main/setup.sh)
```
